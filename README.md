# 🌤️ Condições do Tempo

Um site simples e elegante para consulta de previsões meteorológicas em tempo real, utilizando a API do OpenWeatherMap.

## 🚀 Tecnologias Utilizadas

*   **TypeScript** 📘: Tipagem estática para um código mais seguro e robusto.
*   **HTML5 & CSS3** 🎨: Estrutura e estilização com foco em experiência do usuário.
*   **Fetch API** 🌐: Consumo de dados assíncronos da API externa.
*   **OpenWeatherMap API** ☁️: Fonte de dados meteorológicos globais.

## ✨ Funcionalidades

- [x] **Busca por Cidade**: Digite o nome de qualquer cidade para obter o clima atual.
- [x] **Dados em Tempo Real**: Temperatura arredondada e localização exata.
- [x] **Ícones Dinâmicos**: Visualização do estado do tempo (sol, nuvens, chuva, etc.).
- [x] **Feedback Visual**: Animações de entrada (`fadeIn`) ao carregar novos dados.
- [x] **Tratamento de Erros**: Alertas para cidades não encontradas ou entradas inválidas.

## 🛠️ Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/condicoes-do-tempo.git
   ```

2. **Instale as dependências (TypeScript):**
   Certifique-se de ter o Node.js instalado.
   ```bash
   npm install
   ```

3. **Compile o código TypeScript:**
   ```bash
   npx tsc
   ```
   *Os arquivos compilados serão gerados na pasta `/js` conforme configurado no `tsconfig.json`.*

4. **Abra o projeto:**
   Basta abrir o arquivo `index.html` no seu navegador de preferência.

## 📝 Estrutura de Pastas

*   `/ts`: Contém os arquivos fonte em TypeScript.
*   `/js`: Contém os arquivos JavaScript compilados.
*   `index.html`: Página principal da aplicação.
*   `tsconfig.json`: Configurações do compilador TypeScript.
